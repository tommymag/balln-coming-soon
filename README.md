# BALLN Coming Soon

Temporary one-page billboard for balln.shop.

## Project files

- `index.html` — page structure
- `styles.css` — layout, colors, glow, and responsive styling
- `script.js` — optional JavaScript
- `assets/balln-coming-soon.png` — main billboard artwork

## Local preview

Open the folder in Visual Studio Code and click **Go Live**.

## Live-site goal

Publish this billboard to balln.shop with:

- no navigation
- no buttons
- no external links
- no marketplace access

The full marketplace is maintained separately in the `balln-marketplace` project.
